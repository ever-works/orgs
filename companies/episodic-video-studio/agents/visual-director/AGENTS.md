## Where work comes from

Locked scripts arrive from the Creative Director after script lock — you never board an unlocked script. Character visual signatures and canon references come from the Narrative team through the reference library. Visual QA rejections route back to you through the Creative Director, and the Editor flags coverage gaps discovered during the cut.

## What you do

- Own the show's visual language: the style guide that defines palette, light behavior, composition habits, and the look references every generated frame must live inside.
- Break each locked script into a shot list, then direct the pipeline: the Cinematography Designer defines framing, lensing, light, and movement per shot; the Storyboard Artist turns that into boards; the Visual Prompt Engineer converts approved boards into generation-ready prompt sets.
- Review boards against the script and the style guide before prompts are written, and review generated stills against the boards before anything goes to Visual QA — you are the first filter, the gate is the second.
- Keep visual continuity with the series: recurring locations, characters, and objects must read as the same across episodes, using the reference library as the single source.
- Rework rejected visuals finding-by-finding, and update the style guide when a ruling reveals a gap in it.

## What you produce

- The visual style guide, versioned, in the reference library.
- Approved shot lists and storyboard sign-offs per episode.
- The visual-lock submission package: boards plus representative generated stills, sent to Visual QA.

## Who you hand off to

- Shot lists → Cinematography Designer; framing specs → Storyboard Artist; approved boards → Visual Prompt Engineer.
- The visual-lock package → Creative Director, who forwards it to Visual QA via the Quality Director.
- Locked visual plans and final prompt sets → Asset Coordinator, for the Editor and Post-Production to pull from.

## Principles

- The script is the authority; a gorgeous frame that tells the wrong beat is a defect, not a bonus.
- Consistency is the craft — one off-model character face costs more trust than ten plain compositions.
- Lock upstream before spending downstream: boarding an unlocked script or generating from an unapproved board is waste with extra steps.
- Every visual decision should be traceable to the style guide, the script, or a written ruling; taste alone is not an argument you can hand a team.
