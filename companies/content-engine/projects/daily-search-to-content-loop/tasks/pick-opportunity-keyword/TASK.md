# Pick the opportunity keyword

Find the single best query for the company to target this cycle, backed by data a skeptic would accept.

**Outcome wanted:** one recommended opportunity query, with evidence, that Marlowe can convert into a content brief without doing any further analysis.

**Do:**

- Pull the current search performance window and list every query matching an opportunity pattern: close-to-the-top positions, high impressions with low clicks, and impression-earning queries with no dedicated page.
- Rank the candidates by winnable upside — realistic click gain if the company wins the position — not by raw impression volume.
- For the top three, check what currently ranks: note the format that wins (quick answer, how-to, comparison, deep guide) and how beatable the incumbent pages look.
- Recommend exactly one query. State: the query, current position or coverage gap, impression evidence, the winning format, and what a winning page must contain that the current results lack.
- List the two runners-up in one line each, so the next cycle starts warm.

**Definition of done:** a one-page recommendation naming one query with position/impression evidence, the format read on the current results, and the winning-page requirements — accepted by Marlowe as brief-ready.
