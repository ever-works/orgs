# Figure Preparation

Turn correct charts into publication-ready figures that meet a journal's or report's standard and carry the paper's argument.

## Procedure

1. **Confirm the message and placement.** Know which claim in the text each figure supports and where it sits, so panel order follows the narrative.
2. **Plan the layout.** Group related panels into one figure; label panels A, B, C in reading order. Size for the target column or page width so nothing is rescaled later and blurred.
3. **Standardize style across panels.** One typeface, consistent font sizes, aligned axes, and a shared color mapping for any variable that recurs across panels.
4. **Apply a colorblind-safe palette** and add redundant encoding (markers, line styles, hatching). Verify legibility in grayscale.
5. **Complete every label.** Axis titles with units, tick labels, legends, significance markers with their test named, and scale bars where relevant. No panel should require the caption to be intelligible.
6. **Represent uncertainty** consistently — same error convention across panels, defined in the caption (SD, SEM, CI, or n).
7. **Export at publication resolution** in the required format; keep vector where possible for line art. Preserve a source-data link for each panel.
8. **Proof at final size.** Shrink to print dimensions and check that the smallest text and thinnest line are still readable.

## Quality bar

- Panels share a consistent visual language; nothing looks pasted from two tools.
- Every panel is self-explanatory with complete units and labels.
- Colorblind-safe and grayscale-legible.
- Each panel links back to the analysis artifact that produced it.

## Output format

A composed figure (multi-panel where relevant) at publication resolution, a caption stating each panel's message and its uncertainty convention, and a source-data note mapping panels to artifacts.
