# Options Comparison

Compare alternatives so the comparison decides. The failure mode this prevents: the feature-matrix that scores everything, weights nothing, and gets ignored while the decision is made on vibes anyway.

## Steps

1. Fix the option set with the requester: the named alternatives, plus always **"none of these / keep the status quo"** as a real row. If a plausible option is being excluded, record why.
2. Derive criteria from the decision, not from the options' brochures: what must the winner do, what would disqualify a candidate, what matters but is negotiable. Cap at 5-8 criteria; beyond that, weights become noise.
3. Split criteria into **gates** (binary, disqualifying — a license conflict, a hard cost ceiling) and **scored dimensions**. Run gates first and eliminate; do not score the dead.
4. Weight the scored dimensions before scoring any option — weights set after scoring drift toward the evaluator's favorite. Get the requester to sign the weights.
5. Score each survivor per dimension with an evidence note: what was checked, not what was felt. Distinguish observed ("we tested it"), reported ("their docs / a practitioner says"), and claimed ("their marketing says"). Uncertain scores get a range, not a midpoint.
6. Compute the weighted result, then immediately stress it:
   - Flip any reported/claimed score to its pessimistic end — does the winner change?
   - Would a plausible weight shift (±20% on the top two weights) change the winner?
   If either flips the result, the comparison is genuinely close: say so, and name the single check that would settle it.
7. Write the recommendation as a sentence with a condition: "Choose A, unless X turns out true, in which case B." Include the runner-up's losing margin so the reader knows how contested the call was.

## Quality bar

- Status quo appears as a row and gets scored honestly.
- Every score cites its evidence class; a column of unmarked scores is opinion in a costume.
- The stress test is shown, not just claimed.

## Output format

Option set with exclusions → gate results → signed weights → scored matrix with evidence notes → stress-test outcome → conditional recommendation with losing margin.
