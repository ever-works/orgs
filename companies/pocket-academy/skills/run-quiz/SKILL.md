# Run Quiz

The quiz is where the academy learns whether the lesson worked. Three questions, run conversationally, scored honestly, and always followed by a next step — never a dead end.

## Designing the three questions

Bind each quiz to one lesson and test three different things:

1. **The concept** — can the learner state or recognize the core idea in different words than the lesson used?
2. **The application** — given a new concrete case (not the lesson's example), can they apply it?
3. **The misconception** — presented with the common mistake the lesson corrected, do they catch it?

Rules per question: four options labelled A–D, exactly one correct, distractors that are plausible wrong beliefs rather than obvious jokes. Never test recall of the lesson's exact wording, and never make the correct option the longest one.

## Running it

- Announce the quiz in one friendly line, then send question 1 alone. Wait for the answer before sending question 2. One question per message, always.
- Accept relaxed answers: "a", "A", or the option's text all count. An off-topic reply gets one gentle re-prompt; a second gets the quiz paused, not failed.
- Acknowledge each answer neutrally as it arrives; reveal correctness at the end unless the learner asks sooner.

## Scoring and what happens next

- **3/3** — congratulate specifically ("you caught the misconception most people miss"), then suggest the next topic in their sequence.
- **2/3** — pass. Congratulate, give the one-line correction for the missed question, proceed on the normal review ladder.
- **0–1/3** — miss. Encourage without condescension, correct each wrong answer in one line, and pull the topic's review forward to tomorrow. Never repeat the whole lesson unprompted; offer it.

Record per learner: score, which distractor each wrong answer chose, and timestamp. The chosen distractor is the diagnostic — it says what the learner actually believes.

## Reading the aggregate

A question most learners miss indicts the material, not the learners: either the question is ambiguous (answers scatter across distractors) or the lesson failed to teach (answers cluster on one wrong belief). Distinguish which, and flag it to the learning lead with the answer distribution as evidence.
